# Voter-DAPP-in-Stellar
# 🗳️ Stellar Soroban Voting DApp

## 📌 Project Description
This project is a decentralized voting application built on the Stellar Soroban smart contract platform. It enables users to participate in transparent and secure voting directly on-chain without relying on any centralized authority.

The goal of this project is to demonstrate how blockchain technology can be used to build trustless voting systems.

---

## ⚙️ What It Does
- Allows creation of proposals (polls)
- Enables users to vote on proposals
- Prevents double voting using wallet authentication
- Stores all voting data securely on-chain

---

## ✨ Features
- 🔐 Secure voting using Stellar wallet authentication (e.g., Freighter)
- 🧾 Immutable vote records stored on blockchain
- 🚫 One wallet = One vote per proposal
- 📊 Real-time vote count retrieval
- ⚡ Lightweight and gas-efficient Soroban contract

---

## 🧱 Tech Stack
- **Blockchain:** Stellar Soroban
- **Smart Contract Language:** Rust
- **Frontend (optional):** React / Next.js
- **Wallet Integration:** Freighter Wallet
